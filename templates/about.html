It is a Stock Image Site for finding all kinds of images.

Why It's Made?

It started with a need for a site where people who can find Images for free.

It's entire codebase is written in HTML, CSS, JavaScript, Ajax and Python. Frameworks like Bootstrap 4 and Flask are used. It uses Masonry Layout for displaying Images, CroppieJS for changing profile picture, MomentJS for working with date and time on the Frontend, MongoDB for storing everything and uses EverPixel for keywording images which is used for searching images.
